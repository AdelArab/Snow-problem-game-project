Task 2 Snapshot - Snow Problem

This version includes:
- Task 1 setup: board, pieces, level 1, image-based GUI.
- Task 2 movement: mouse-click selection, direction clicking, large/small snowballs sliding, move counter, screen updates, and game over when a snowball leaves the board.

This version intentionally does not include Task 3 stacking, Task 4 high scores/all levels, or Task 5 solver.

Run from this SnowProblem folder:
  javac -d out $(find src -name "*.java")
  java -cp out snowproblem.Main
